# FLAMES (Dynamic UI)

This folder includes:

- `index.html` + `style.css` + `app.js`: a beautiful, animated FLAMES web UI (works offline)
- `flames.py`: the original Python CLI version

## Run the UI

Option A (simplest): open `index.html` in your browser.

Option B (recommended): run a local server so everything loads the same in every browser:

```bash
python -m http.server 8000
```

Then open:

- `http://localhost:8000/index.html`

## Run the Python version

```bash
python flames.py
```
